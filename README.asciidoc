CSharpCodePortfolio é um portfólio executável de tutoriais C# e .NET 10.
O projeto organiza conceitos práticos em exemplos de console com menu interativo, explicações guiadas, evidências de execução e testes automatizados nos pontos mais relevantes.

== Proposta

O repositório foi estruturado para servir como material de estudo, demonstração técnica e referência de implementação.
Em vez de apenas listar trechos de código, cada tutorial pode ser executado pela CLI raiz e apresenta o conceito em uma interface de console baseada em Spectre.Console.

O catálogo principal inclui 28 tutoriais sobre assincronismo, sockets, pipes, reflexão, validadores, repository pattern, EF Core InMemory, dependency injection, testes, HTTP, MassTransit, ASP.NET Core, Razor Pages, RavenDB, Identity, OpenID, SMTP e paralelismo.
O projeto também integra a área `EFCore10`, que possui menu próprio para exemplos dedicados de Entity Framework Core 10.

== Requisitos

* .NET SDK 10.0.109 ou superior compatível com `global.json`.
* Terminal com suporte a ANSI para a melhor experiência visual.
* Git, caso queira clonar e evoluir os tutoriais localmente.

== Como executar

[source,bash]
----
git clone https://github.com/darthlinuxer/CSharpCodePortfolio.git
cd CSharpCodePortfolio
dotnet build CSharpCodePortfolio.slnx
----

Abra o menu interativo:

[source,bash]
----
./run-menu.sh
----

No Windows:

[source,bat]
----
run-menu.bat
----

Também é possível chamar a CLI diretamente:

[source,bash]
----
dotnet run --project src/CSharpCodePortfolio.App -- menu
dotnet run --project src/CSharpCodePortfolio.App -- list
dotnet run --project src/CSharpCodePortfolio.App -- run parallelism
dotnet run --project src/CSharpCodePortfolio.App -- run efcore10
----

== Comandos principais

[cols="1,2", options="header"]
|===
|Comando |Descrição

|`menu`
|Abre o menu interativo inicial.

|`list`
|Lista todos os tutoriais executáveis descobertos pelo host raiz.

|`run <id-or-slug>`
|Executa um tutorial pelo identificador numérico ou pelo slug.

|`run efcore10`
|Abre o menu da área dedicada a Entity Framework Core 10 e retorna ao fluxo principal ao sair.
|===

== Arquitetura

O repositório usa uma solução raiz `CSharpCodePortfolio.slnx` e uma estrutura simples em `src/` e `tests/`.
O host `CSharpCodePortfolio.App` descobre tutoriais por metadados, exibe o menu inicial e executa cada tutorial pelo id ou slug.
Os projetos `CSharpCodePortfolio.Shared` e `CSharpCodePortfolio.Tutorials.Abstractions` concentram a UI compartilhada e o contrato mínimo de descoberta.

Cada tutorial vive em seu próprio projeto `net10.0`, o que mantém os exemplos isolados e facilita a leitura.
Quando o conceito precisa de verificação automatizada, há um projeto de teste correspondente em `tests/`.

== Qualidade

Use os comandos abaixo antes de abrir alterações:

[source,bash]
----
dotnet build CSharpCodePortfolio.slnx
dotnet test CSharpCodePortfolio.slnx
dotnet run --project src/CSharpCodePortfolio.App -- list
----

O objetivo do projeto é manter exemplos pequenos, executáveis e claros.
Novos tutoriais devem preservar a experiência do menu, apresentar explicações objetivas em português do Brasil e evitar dependências que não sejam necessárias para ensinar o conceito.
